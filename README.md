# 清肺排毒汤

**服用方法**：水煎服。每天一付，早晚两次（饭后四十分钟），温服，三付一疗程。

**注意事项**：此方适用于新冠病毒感染的肺炎轻型、普通型、重型患者，也可用于流感患者。
